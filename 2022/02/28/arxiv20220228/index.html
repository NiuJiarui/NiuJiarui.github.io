Monte Carlo simulations of multiple populations in globular clusters: constraints on the initial size of the second generation from binary stars

我们展示了对拥有两代恒星的球状星团的蒙特卡罗模拟调查的结果，其中包括两个群体中大部分（fb = 50%）的原始双星。 考虑到潮汐场、二体弛豫、恒星演化和三/四体相互作用的影响，在哈勃时间内跟踪两个星族的动态演化。 幸存双星的比例，一旦考虑到观测偏差和不确定性，将与在真实球状星团中执行的可用径向速度时间序列进行比较，并用于限制第二代的初始空间浓度。 第二代双星的比例似乎只取决于总星团质量与第二代初始大小之间的比率，该比率决定了该恒星种群范围内的平均速度分散。
尽管存在各种不确定性，我们发现只有假设第二代的初始浓度很高（r∼0.1（M/106M⊙）pc），才能获得观察到的分数。 由于潮汐场对第一代结构特性有直接影响，因此第一代二元部分的演化对潮汐场强度（具有不可忽略的星团轨道偏心率影响）更为敏感。

The simulations have been run using the Monte Carlo code originally presented in （Sollima A., Mastrobuono Battisti A., 2014, MNRAS, 443, 3513）.

A Straight Lightning Bolt?! Observation of a Predicted Macro Dark Matter Signature

最近，在澳大利亚珀斯的视频中捕捉到了四次非常直且非常短暂的闪光，彼此相距不到 0.5 秒。 直线闪电最近被确定为对宏观暗物质（宏）的预测——这是粒子暗物质的一大类替代候选物。 一个足够大的宏观穿过大气层产生直的等离子体柱会产生荧光，或者，在有利于闪电的大气条件下，会产生肉眼可见的非常直的雷击。 其他解释被考虑，但有问题。
